# Style
Set your Style

# Command
## /style
### /style set [Player] [Player's Style]

## /칭호
### /칭호 설정 [플레이어] [해당 플레이어의 칭호]

## /귓 [플레이어] [메시지]
### 작동하지 않는 명령어입니다. 구현이 중단되었습니다.

# & Color Code
You can set player's style with & Color code.

# & 색 코드
당신은 & 색 코드를 이용해 당신의 칭호를 정할 수 있습니다.
ex) &aJaemin-MC

# & Color Code

'0' -> BLACK

'1' -> DARK_BLUE

'2' -> DARK_GREEN

'3' -> DARK_AQUA

'4' -> DARK_RED

'5' -> DARK_PURPLE

'6' -> GOLD

'7' -> GRAY

'8' -> DARK_GRAY

'9' -> BLUE

'a' -> GREEN

'b' -> AQUA

'c' -> RED

'd' -> LIGHT_PURPLE

'e' -> YELLOW

'f' -> WHITE
